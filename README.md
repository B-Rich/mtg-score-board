# Magic The Gathering Score Board

## Install
`npm install -g webpack`

`npm install -g webpack-dev-server`

`npm install`

## Start

Compile files

`webpack`

Start dev server

`webpack-dev-server`

Watch task

`webpack --watch`

To load JSON from a file via ajax

`npm i -g json-server`

`json-server db.json`

### Notes
